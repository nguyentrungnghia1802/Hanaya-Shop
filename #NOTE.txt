1, Tạo project:
composer create-project --prefer-dist laravel/laravel hanaya_shop

2, Chạy laravel: 
php artisan serve --host=0.0.0.0 --port=8000

- Trường hợp lỗi không load được app.css thì chạy thêm một terminal với CMD sau:
npm run dev

3, Tạo file mới:
php artisan make:view layouts.slider

# CMD
4. Reset Migrations
php artisan migrate:reset

5. Run seeder
php artisan db:seed

6. Run database
mysql -u root -p hanaya_shop_demo < .\database\sql\sample_data.sql
